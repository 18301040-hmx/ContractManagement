# 合同管理系统

## 1. 项目启动

### 项目名称

合同管理系统

### 项目概述

系统采用 B/S 结构，服务器端需要配置程序的运行环境（JDK1.8）、安装 ORACLE 数据库， 用户端只需要通过浏览器就可以访问该网站。
